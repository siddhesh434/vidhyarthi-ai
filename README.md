# Vidhyarthi-AI

The Open NCERT Tutor RAG Pipeline built on Databricks for the Bharat Bricks Hackathon.

## Features
- **Delta Lake Integration:** Vector embeddings and text chunks stored directly in Databricks.
- **Sarvam AI Engine:** Natively connects to `sarvam-105b` for blazing fast, localized Indian-language responses.
- **Asynchronous User Memory:** Background student profiling using multi-threading.
- **Native Browser Speech:** Allows students to talk directly to the system.
